# Interface Overview

A quick guide to the Hero's Guild user interface and controls.

## Main Menu

From the main menu you can:
- **New Game** - Start a fresh guild
- **Continue** - Load your saved game
- **Settings** - Audio, graphics, controls
- **Quit** - Exit the game

---

## Guild Screen

The Guild Screen is your home base. It's divided into several areas:

### Top Bar

- **Gold** - Your current funds
- **Day** - Current game day
- **Guild Rank** - Your reputation rank (F-S)
- **Settings** - Access options

### Hero Panel (Left)

Lists all your heroes:
- Portrait and name
- Class icon
- Level
- Current state (Ready, Injured, etc.)
- Mood indicator

Click a hero to see details.

### Facility Panel (Center)

Shows your guild layout:
- Click facilities to access them
- Upgrade indicators show available upgrades
- Active crafting shows progress

### Info Panel (Right)

Context-sensitive information:
- Selected hero details
- Facility information
- Mission details

---

## Hero Details

When you select a hero:

### Stats Tab

- Base stats (STR, DEX, INT, VIT, LCK)
- Derived stats (HP, Mana, Initiative)
- Level and XP bar
- Paragon points (if level 100+)

### Equipment Tab

- All 8 equipment slots
- Drag items to equip
- View item stats
- Socket management

### Skills Tab

- Active skills
- Passive tree
- Skill gem management
- Proficiency levels

### Social Tab

- Relationships with other heroes
- Mood and needs
- Traits
- Recent events

## Combat Screen

During supervised expeditions:

### Turn Order (Top)

Shows initiative order:
- Current actor highlighted
- Heroes vs enemies
- Status effects visible

### Battle Area (Center)

- Your party on left
- Enemies on right
- Health bars above each
- Status effect icons

### Action Bar (Bottom)

- Available actions (Attack, Skills, Defend, Flee)
- Skill cooldowns shown
- Mana costs displayed

### Combat Log (Side)

- Damage numbers
- Skill usage
- Status effects
- Hero dialogue

### Command Points

In supervised mode:
- Command point display
- Available interventions
- Emergency retreat option

## Inventory Management

### Guild Vault

Central storage for all items:
- Categories (Weapons, Armor, etc.)
- Sort options (Rarity, Type, Level)
- Search function

### Item Actions

- **Equip** - Assign to hero
- **Sell** - Convert to gold
- **Salvage** - Get materials
- **Enchant** - Add magic

## Quick Tips

1. **Hover for tooltips** - Most elements have explanations
2. **Right-click for options** - Context menus on items/heroes
3. **Drag and drop** - Equipment, party formation
4. **Watch the log** - Combat details, events
5. **Check notifications** - Red dots indicate attention needed
